// 각종 편의 기능을 모아둔 모듈
// deprecated와 promisify가 자주쓰임

// deprecated : 
- 코드를 만들었는데 잘못 만듬 근데 수정해야하는데 그렇다고 현재 코드를 지울 순 없음
ex) JS는 ES6모듈이 나왔는데 Commonjs모듈을 바꿔야함 근데 이걸 수정하면 Commonjs 사용했던 사람들이 에러가 나게 됨
- 이걸 쓸 때마다 경고창을 출력하게 해줌
- "버전이 달라졌으니 이걸 사용하지 말아달라 이렇게"

// promisify : 
- node에는 프로미스 지원 안하는게 몇개 있음, 아직 콜백함수 쓰는 곳이 있음
- 그럴 때 저걸로 감싸주면 Promise로 then을 붙여줄 수 있음
- 그래서 await을 붙일 수 있음
- 1가지 조건 : 콜백이 (error, data) => {} 형식이어야 함(노드는 저런 형식으로 되어 있음)
